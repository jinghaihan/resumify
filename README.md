# Resumify

Block-based resume builder with AI and PDF export.

  <img src='./assets/screenshot.png' alt="screenshot" />
</p>

## Features

- Block-based editing: build resumes by combining reusable content sections.
- Drag-and-drop layout: reorder blocks quickly to iterate on structure.
- AI assistance: generate or refine resume content.
- Theme system: switch templates and visual styles.
- Markdown support: rich text formatting in supported content fields.
- PDF export: produce printable resumes from the same source data.
- Multi-language architecture: built with i18n-ready app structure.
